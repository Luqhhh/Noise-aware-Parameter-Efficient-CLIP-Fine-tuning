# L1：cross-fitted 有效模型先验的纯推理期校正

日期：2026-07-20

## 大方向判断

当前一系列训练内标签修正和对比学习均受到“同一个 CLIP 表征确认自身预测”的限制。另一方面，官方测试集明确按 500 类均衡构造，而 A2 保留训练集单类数量为 36--192，模型仍可能保留有效类别先验偏置。WACV 2025 Prior2Posterior（P2P）指出：模型实际学到的先验不等于样本频率，应从模型后验概率估计，并在平衡测试时用 Bayes 形式进行 post-hoc 修正。该方向不修改模型、不读取测试标签、不做测试时参数适应，也不引入模型融合。

本地 A2 未见验证集提供了高度匹配的压力测试：10,322 张样本中，178 类各 20 张、322 类各 21 张，全部 500 类覆盖，近似严格均衡。因此可以在不构造新划分的情况下检验均衡先验是否改善 J0。

## 冻结方法

1. 固定模型：J0 `best.pt`，不再训练；
2. 固定有效先验来源：I0 重建的 91,195 张训练样本 cross-fitted A2 OOF logits；每张先做 temperature=1 的 softmax，再对样本求均值得到 `P_m(y)`；
3. 固定目标先验：500 类均匀分布 `P_t(y)=1/500`；
4. 唯一校正式：`z_corrected = z - log(P_m(y)) + log(P_t(y))`，strength=1，不使用标签调强度；
5. 先缓存一次 J0 未见验证 logits，然后同时计算裸预测与 L1；验证标签只在公式完全冻结后用于评分；
6. 本轮不运行 Sinkhorn、不使用验证或测试批次的预测边际反推先验，避免测试集传导式适配的合规歧义。

## 预注册门槛

1. clean-core micro 相对同一 logits 的裸预测至少 `+0.30pp`；
2. trusted macro 不下降，raw micro 不得下降超过 `0.20pp`；
3. 预测空类数量和预测计数变异系数必须同时下降，证明收益确由类别偏置修正而不是偶然翻转；
4. 任一门槛失败即关门：不扫描 correction strength、softmax temperature 或 prior floor，不进入测试推理，不生成提交包；
5. 只有全部通过，才允许把同一冻结公式审计到 A2/F1 与 flip-TTA 单模型流程。

## 合规与隔离

有效模型先验只来自训练集 cross-fitted 预测；测试集在未来最多进行一次普通前向和固定公式校正，不参与训练、自监督、无监督学习或参数更新。最终仍是一个 CLIP ViT-B/32 的单路预测及确定性后处理。团队仓库保持只读，全部实现与产物在独立工作树中。

## 严格结果

J0 验证 logits 缓存 SHA-256：`138d7f5e880100e3365110ead600c52d8d72dbd13f850a23190c969c257ae607`；I0 OOF logits 完整 SHA-256：`b2bb6f72c0f44961cd1665d81443dc5afe58b264702c1c5e9672e55e399440b3`。

cross-fitted 有效先验的归一化熵为 0.99819、CV 为 0.13968，整体已接近均匀，但极端类别 max/min 为 14.68。固定 P2P 公式改动 220/10,322 个预测（2.1314%），并确实减弱输出数量不均衡：prediction count CV 从 0.22016 降到 0.19473，单类最少预测由 2 升至 8、最多预测由 38 降至 34，空类始终为 0。

但准确性门槛失败：

- clean-core micro：`82.8389% → 82.8112%`，`-0.0277pp`；
- clean-core macro：`82.7479% → 82.7751%`，`+0.0273pp`；
- raw micro：`69.7442% → 69.6764%`，`-0.0678pp`；
- trusted macro：`80.7065% → 80.6706%`，`-0.0359pp`。

该机制成功改善了预测计数的表面均衡，却没有提升样本级真实性，说明当前主要瓶颈不是可由全局类别先验消除的决策偏置。按预注册不扫描 strength/temperature/prior floor，不运行测试集 Sinkhorn、不生成提交包。
