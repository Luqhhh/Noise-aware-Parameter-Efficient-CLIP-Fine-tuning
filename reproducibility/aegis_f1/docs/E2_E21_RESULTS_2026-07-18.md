# E2–E21 实验与提交记录（2026-07-18）

## 1. E2 完整复现

- 配方：冻结 OpenAI CLIP ViT-B/32，CE 5 epoch warm-up，随后 GCE `q=0.5`；pixel MixUp `alpha=0.2, p=0.2`；seed 42。
- 完整运行 50 epoch；最佳为 epoch 44。
- 最佳验证 raw micro：`70.2501%`；raw macro：`70.1974%`。
- 最佳检查点 SHA-256：`50e05d09921a0f9bf852589cae848b926c61892e6952f1082dfa25daae2e3ff6`。
- 该结果与团队报告的本地约 `70.25%` 实质持平；独立工程未修改团队工程。

- 缓存双视图 raw micro：`70.9190% → 71.1031%`（+0.1840 pp）。
- 587 个预测改变；修复 140，破坏 121，净修复 +19。
- trusted macro：`80.4721% → 80.6211%`。
- proxy macro：`79.3075% → 79.4536%`。
- 真实图像在线 mean-logit TTA：`70.8996% → 71.1322%`（+0.2326 pp）。
- 在线 mean-probability TTA：T=1 为 `71.1710%`，T=2 为 `71.1419%`；固定选择 T=1。
- E20 检查点 SHA-256：`4ef590196d52dfcfe13016d2f72ac72cc81114575596edffc8a1945931a2ffa6`。
